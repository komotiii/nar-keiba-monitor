# nar-keiba-monitor

地方競馬 (NAR) の出走表・過去成績・オッズを取得し、条件に合う馬をコンソール上で監視・表示する Python スクリプトです。

## 概要

`netkeiba.com` からリアルタイムにデータをスクレイピングし、独自の閾値（複勝率やオッズ）に基づいた「注目馬」をダッシュボード形式で表示します。

### 主な機能

- **自動巡回**: レースIDを自動生成し、当日の対象競馬場の全レースを時系列で巡回。
- **データ分析**: 出走表から過去5走の成績を取得し、複勝率を算出。
- **リアルタイム監視**: 最新の単勝・複勝オッズを定期的に取得し、画面を更新。
- **ハイライト表示**: 指定した条件（例：複勝率0.4以上、複勝オッズ1.5〜4.0倍）に合致する馬を赤字で強調。
- **音声通知**: 締切3分前と締切時にローカルの音声ファイル（.mp3）を再生してお知らせ。
- **キャッシュ機能**: 取得したレース情報は当日中 `datas/` ディレクトリに保存され、再起動時の負荷を軽減。

## ファイル構成

```text
.
├── main.py              # 実行メインスクリプト
├── config.json          # 設定ファイル（各自作成）
├── requirements.txt     # 依存ライブラリ
└── datas/               # レース情報のキャッシュ（自動生成）
```

## セットアップ

### 1. 依存パッケージのインストール
```bash
pip install -r requirements.txt
```

### 2. 設定ファイルの作成
`config.example.json`（または以下の内容）を `config.json` として保存します。

```json
{
    "thresholds": {
        "win_rate": 0.4,
        "odds_min": 1.5,
        "odds_max": 4.0
    },
    "site_ids": [
        [36, "水沢"],
        [42, "浦和"]
    ],
    "wait_sec": 30,
    "paths": {
        "data_dir": "datas",
        "audio_pic": "path/to/alert1.mp3",
        "audio_ftn": "path/to/alert2.mp3"
    },
    "headers": {
        "User-Agent": "Mozilla/5.0"
    }
}
```

## 使い方

スクリプトを実行すると、自動的に当日の全レースをスキャンし、直近のレースから監視を開始します。

```bash
python main.py
```

### 画面の見方
- **馬名が赤太字**: 設定した「複勝率」と「オッズ範囲」の両方を満たしている注目馬です。
- **カウントダウン**: 次のオッズ更新までの秒数が表示されます。
- **自動遷移**: レース発走時刻が近づくと、自動的に次のレースの監視に移ります。

## 注意事項

- **自己責任**: 本ツールは情報の集計を目的としており、的中を保証するものではありません。馬券の購入は計画的に行ってください。
- **マナー**: サイトへの負荷を避けるため、`wait_sec`（更新間隔）は極端に短く（例: 5秒以下など）設定しないようにしてください。
- **環境**: 音声通知機能には `pygame` を使用しているため、オーディオデバイスが利用可能な環境で実行してください。
- **メンテナンス**: サイトの仕様変更により、スクレイピングができなくなる可能性があります。
